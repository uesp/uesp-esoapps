ESOAPPS contains a variety of Windows applications related to ESO (Elder Scrolls Online)
and created by the UESP (Unofficial Elder Scrolls Pages, www.uesp.net). Contact Dave
at dave@uesp.net for more information.

Unless otherwise stated all code is released under the GPL v2 (redistributed library code
is released under its original license).

	\common\ 
		Common code and redistributed libraries
	\EsoExtractData\
		Windows command line program to extract data from various ESO 
	    data files (MNF, DAT, ZOSFT, LANG, etc...).
	\uespLogMonitor\
		Windows GUI program used to automatically monitor and upload data collected 
		from the uespLog add-on program for ESO.
